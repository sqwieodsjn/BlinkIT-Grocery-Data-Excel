# BlinkIT Grocery Sales Analysis Dashboard

## Project Overview

This project focuses on analyzing BlinkIT grocery sales data using **Advanced Excel** to generate business insights, KPI tracking, and interactive dashboards. The analysis helps understand customer purchasing trends, outlet performance, product category sales, and operational metrics.

The project demonstrates practical skills in:

* Advanced Excel
* Data Cleaning
* Pivot Tables
* Pivot Charts
* KPI Dashboard Creation
* Business Data Analysis
* Data Visualization
* Reporting & Insights
